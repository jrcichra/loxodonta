# Loxodonta
A social media platform for privacy-focused mammals

## Purpose/Goals
+ Self-hosted social media ( maybe IPFS support at some point?)
+ Easier to set up than Mastodon
+ Keep it simple

## Tech Stack
+ Vue / Nuxt.js
+ GraphQL
+ MariaDB
+ Node.js
+ Docker (easy deployments)
